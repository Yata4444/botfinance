# Фінансовий трекер

## Мета проєкту
Розроблено консольного чат-бота мовою Python, який допомагає студентам вести облік особистих фінансів: контролювати встановлений бюджет, фіксувати витрати за категоріями та відстежувати залишок коштів, отримуючи попередження у разі перевищення бюджету.

## Опис логіки та алгоритму роботи
Бот працює у форматі безперервного циклу , очікуючи на команди користувача через консоль. 
1. **Запуск**: Програма перевіряє наявність файлу бази даних. Якщо він є — завантажує дані, якщо ні — створює порожній шаблон бази даних. Виводиться вітання.
2. **Обробка запитів**: Введені команди аналізуються через структуру `if-elif-else`.
3. **Виконання дій**: Залежно від команди викликається відповідна функція (додавання витрати, обчислення залишку, фільтрація).
4. **Збереження**: Після кожної операції, що змінює стан даних (наприклад, встановлення бюджету або додавання витрати), інформація автоматично перезаписується у файл бази даних.
5. **Завершення**: При введенні команди "вийти" цикл переривається і програма завершує роботу.

## Структура даних
Дані зберігаються у текстовому файлі `finance_data.json` у форматі JSON. Це словник (dictionary) з двома головними ключами:
* `budget` (float) - загальна сума встановленого бюджету.
* `expenses` (list) - список словників, де кожен словник є окремою витратою.
Кожна витрата містить:
  * `amount` (float) - сума.
  * `category` (string) - категорія витрати.
  * `date` (string) - дата у форматі ДД.ММ.РРРР.
  * `comment` (string) - додатковий коментар користувача.
